# GiroERP - Frontend

## Tecnologias
- React 18
- TypeScript
- Tailwind CSS
- Vite
- React Router DOM
- Axios
- React Hook Form
- Recharts

## Como executar

```bash
# Instalar dependências
npm install

# Executar em desenvolvimento
npm run dev

# Build para produção
npm run build

# Visualizar build
npm run preview
